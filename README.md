## Pee Break (Canvas)

A tiny retro/pixelated JavaScript canvas toy.

### How to run

- Easiest: open `index.html` in your browser.

If your browser blocks local scripts, run a local server:

```bash
cd pee-break
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

### Deploy to GitHub Pages

Once you create a GitHub repo for this project:

- Push these files to the repo (root should contain `index.html`, `style.css`, `game.js`).
- In GitHub: **Settings → Pages**
  - **Source**: Deploy from a branch
  - **Branch**: `main` (or `master`) / **root**
- Your URL will be:
  - `https://<org-or-user>.github.io/<repo>/`

### Controls

- Click / hold on the screen to pee (aim follows your cursor)
- `R` to reset

### Goal

No objective — just a calm sunset canyon and the aesthetics of the stream.
